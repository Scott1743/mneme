# LLM Wiki topic

This topic page bundles the two foundational concepts that Mneme is built
on. Each concept carries its own definitions, history, and external
references; this page is the entry point that the agent consults when
the user asks broad questions about Mneme's intellectual lineage.

## Concepts

- [LLM Wiki](/concepts/llm-wiki.md) — Karpathy's idea of a persistent,
  agent-maintained knowledge wiki; the *why* of Mneme.
- [OKF](/concepts/okf.md) — Open Knowledge Format v0.1: markdown +
  YAML frontmatter, three conformance rules; the *how* of Mneme.
