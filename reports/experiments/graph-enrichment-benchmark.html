Graph enrichment retrieval benchmark

A controlled ablation of deterministic Graph, agent enrichment, and global FTS5 fusion on one 142-page Markdown bundle. The benchmark contains 72 construction-aware answerable queries and 8 synthetic no-answer controls.

Enrichment changes Graph nDCG@10 by +0.494 and hybrid by +0.341. The H1-L1 paired difference is +0.568; confidence intervals and family breakdowns below define the valid scope of that result.

Overall retrieval quality

Points are mean binary nDCG@10; horizontal lines are query-bootstrap 95% confidence intervals (10,000 resamples). No-answer controls are excluded from ranking metrics.

Classic metric profile

Precision@10 uses a fixed denominator of 10. Accuracy is top-1 relevance, not classification accuracy over documents. Recall and F1 are macro-averaged over answerable queries.

Query-family response

The family split is essential: entity and relation qrels are derived from the frozen extraction manifest and measure mechanism coverage, not independent general-search quality.

Paired effects

Positive values favor the second system. Intervals crossing zero do not establish a stable directional effect on this diagnostic set.

Latency

Warm in-process measurements; each query is repeated 5 times. They describe this local machine and are not service-level benchmarks.

Methods and limits

Corpus. One private 142-page Feishu Markdown export. Export pairs with identical bodies (foo.md/foo--2.md) are treated as one document equivalence class in qrels and ranked candidates.

Metrics. nDCG uses binary relevance and logarithmic rank discount. Recall is macro-averaged per answerable query; Hit records any relevant top-10 result; MRR uses the first relevant top-10 rank. No-answer controls are excluded and reported as FPR.

Systems. L1 is global FTS5. G0 derives only pages, tags, and Markdown links. G1 adds the frozen agent extraction manifest. H0/H1 use the production Graph + global FTS union.

Labels. Entity, context, and relation qrels are deterministically sampled from the extraction manifest. They are suitable for enrichment ablation, but share construction provenance with G1 and must not be treated as independent human relevance judgments.

Known boundary. This report does not compare L2, answer synthesis, citation correctness, or independent user questions. A separate double-annotated benchmark is required for those claims.
